This demo showcases real-time audio processing using CuteDSP's JavaScript implementations. It applies multiple effects to your microphone input with individual enable/disable controls.

Available Effects:

  • Filter: Biquad low-pass filter with adjustable cutoff and resonance
  • LFO Modulation: Low-frequency oscillator that modulates the filter cutoff
  • Delay: Feedback delay effect with adjustable time and feedback
  • Distortion: Soft-clipping distortion with drive and amount controls
  • Reverb: Schroeder reverb with adjustable mix and decay
  • Chorus: Multi-voice chorus effect with modulation depth control

Features:

  • Real-time processing with AudioWorklet API
  • Individual effect enable/disable toggles
  • Visual waveform display
  • Performance monitoring and optimization
  • Built with JavaScript DSP implementations for reliability
